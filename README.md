# Avalin Ulagam 🌸

A cinematic animated portfolio is crafted by blending cartoon-style animation frames with interactive web experiences.
The intro sequence was created by designing frame-based animations, converting them into motion visuals using EZGIF,Flow,and integrating them seamlessly into the website flow.

Built with immersive transitions, cinematic interactions, and responsive modern UI experiences.

## ✨ Features

* Cinematic animated intro
* Cartoon-integrated portfolio experience
* Smooth frame-based transitions
* Interactive storytelling UI
* Responsive mobile & desktop layouts
* Framer Motion powered animations
* Immersive project showcase
* Contact integration using EmailJS

## 🛠 Tech Stack

* React
* Vite
* Framer Motion
* Tailwind CSS
* EmailJS

## 🎬 Animation Workflow

* Cartoon scenes designed frame-by-frame
* Frames converted into animated sequences using EZGIF
* Integrated directly into website transitions
* Cinematic flow between intro and portfolio sections

For the best cinematic experience, use a PC or laptop.
